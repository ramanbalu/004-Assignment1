Before you start executing the LoadData.R
ensure that you change the current working diretory to the one where you have the data.

Set the working directory to your working directory 

Example: (Change "C:/Project"  to your working directory)
setwd("C:/Project")

After making changes to the working directory in the code save the file.

Step 2: Execute Plot1.R   -- Plot1.PNG will be saved in your current Working Directory
